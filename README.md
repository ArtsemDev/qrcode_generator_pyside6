## QRCode Generator with PySide6

### CREATING ENVIRONMENT
`python -m venv venv`

`venv\Scriptis\activate` - WINDOWS

`source venv/bin/activate` - UNIX

### INSTALLATION DEPENDENCIES
`pip install -r requirements.txt`

### RUN
`python app.py`

### COMPILATION
`pyinstaller --onefile --noconsole app.py`
